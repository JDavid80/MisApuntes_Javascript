Operadores Relacionales

=== .... Estrictamente igual

!== .... Distinto. Devuelve false si no son estrictamente iguales, incluso el tipo de dato

== .... Igual, independientemente del tipo de dato

!= .... Distinto

> .... Mayor que

< .... Menor que

>= .... Mayor o igual que

<= .... Menor o igual que

En el siguiente ejercicio veremos mas a fondo estos operadores
